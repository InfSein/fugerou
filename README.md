# fugerou

自用魔改版 [Kagerou](https://github.com/hibiyasleep/kagerou)。

## 特色功能

### 本地化优化
* 默认语言从韩语调整为中文
* 完善中文本地化文本

### 易用性优化
* 增加设置项“自定义自己的名称”
* UI缩放最大值从2倍提升至3倍

### 修复
* 修复设置了缩放的场合，设置页面元素会变得过于巨大的问题
* 修复一些大小/边距细节问题

## 使用方法

1. 在 `OverlayPlugin` 中新建悬浮窗，类型选择 `自定义悬浮窗`；
2. 将悬浮窗地址设置为 `https://infsein.github.io/fugerou/overlay/`。
